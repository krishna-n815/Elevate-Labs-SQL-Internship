# Task3
# Basic SELECT Queries in SQL

## Objective
To practice and demonstrate basic SQL `SELECT` queries for extracting data from one or more tables using filtering, sorting, and limiting techniques.

## Tools
- MySQL Workbench

## Features Demonstrated
- SELECT all columns (`SELECT *`)
- SELECT specific columns
- Filtering with:
  - `WHERE`
  - Logical operators: `AND`, `OR`
  - Pattern matching: `LIKE`
  - Range filtering: `BETWEEN`
- Sorting results with `ORDER BY`
- Limiting results with `LIMIT`

## SQL Queries Included
- Select all data from the table.
- Select specific columns.
- Apply `WHERE` with different conditions.
- Use `AND`, `OR` for multiple conditions.
- Search with `LIKE` for patterns.
- Filter using `BETWEEN` for ranges.
- Sort records using `ORDER BY` (ascending/descending).
- Limit the number of output rows using `LIMIT`.

## Files Included
- `task_3.sql` — SQL script file with:
  - Table creation
  - Data insertion
  - SELECT queries in one line each
- `README.md` — This file

## How to Run
1. Open **MySQL Workbench**.
2. Open the task_3.sql` file.
3. Run the script to create the table, insert data, and execute SELECT queries.
4. View the output of each query in the result pane.

## Mini Guide (Mints)
1. Use `SELECT *` and specific columns to retrieve data.
2. Apply `WHERE` with `AND`, `OR`, `LIKE`, `BETWEEN` to filter data.
3. Sort the data using `ORDER BY` and limit rows with `LIMIT`.
